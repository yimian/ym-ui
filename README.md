# ym-ui

## Introduction

[ym-ui](https://ymui.yimian.com.cn) is a front-end solution.

## Getting started

```shell
# clone the project
git clone https://github.com/yimian/ym-ui.git

# enter the project directory
cd ym-ui

# install dependency
npm install

# develop
npm run serve
```
